# Front-End-Gerenciador-de-tarefas
Front-End do Gerenciador de Tarefas, desenvolvido para permitir a criação, visualização, edição e exclusão de tarefas de forma simples e intuitiva. A aplicação consome uma API REST, garantindo integração com o back-end e atualização dinâmica dos dados. Foco em organização, produtividade e boa experiência do usuário.
